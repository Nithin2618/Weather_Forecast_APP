# Weather_Forecast_APP

This is a simple weather application built with **Vite**, **TailwindCSS**, and **PostCSS** to provide real-time weather data for various locations. The app allows users to search for a city and view the current weather conditions.

## Features
- Search for weather by city
- Displays temperature, humidity, wind speed, and more
- Responsive design with TailwindCSS

## Technologies Used
- **Vite** for fast development
- **TailwindCSS** for styling
- **PostCSS** for CSS preprocessing

## Steps to Run the App

### 1. Clone the repository

Start by cloning the project to your local machine:

```bash
git clone https://github.com/Nithin2618/Weather_Forecast_APP.git
cd Weather_Forecast_APP
```
### 2. Install dependencies
Install TailwindCSS, PostCSS, Autoprefixer and Vite

```bash
npm install -D tailwindcss@3 postcss autoprefixer

npm install vite
```

### 3. Run the app

```bash
npm run start
```
